# gsm-test

## First run gsm-test-backend project

### Run docker

```bash
$ docker-compose up
```

### Project local setup

```bash
$ yarn
```

### Compile and run the project local

```bash
# development
$ yarn start
```


## Run gsm-test-frontend project

### Run docker

```bash
$ docker-compose up
```

### Run it tests
```bash
$ yarn cypress open
```


### Project local setup

```bash
$ yarn
```

### Compile and run the project local

```bash
# development
$ yarn start
```
